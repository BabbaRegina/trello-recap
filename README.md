# trello-weekly-recap

```
npm i -g trello-weekly-replay
```

## configuration

Get an API Key and Token from Trello.

Set the following environment variables:

```
TRELLO_API_KEY
TRELLO_API_TOKEN
```


## usage

```
trello-weekly-recap [BOARD_NAME] [SINCE_DATE]
```

### options

#### BOARD_NAME

Specify the name of the board.
